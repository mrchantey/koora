# Troubleshooting


## Memory

The framework currently does not support dynamic wasm module memory `memory.grow etc`. This means that sufficient memory must be allocated when compiling. This can be done by using the `--initialMemory` flag. For the examples on this site we are currently setting this to 4, ie `--initialMemory 4`.
